# Peaceful Home
A mod for Elin that prevents hostile creatures from randomly appearing on your property.

Effects (only in your property):
* Prevents the periodic spawning of enemies.
* Forces random wildlife to be nonhostile animals.
* Suppresses the effects of NPCs failing to read spellbooks.

Changelog:
* 0.0.1a: Initial alpha build.
* 0.0.2a: Changed hostile spawn blocking so that bases don't get overrun with infinite prostitutes. Prevented NPCs from accidentally summoning enemies when reading books.
* 0.0.3b: Fixed hostile spawn blocking. Initial public release.

Credits:
* Based on Omega's mod Safe Home (https://steamcommunity.com/sharedfiles/filedetails/?id=3385438542), which completely prevents any and all spawning.
